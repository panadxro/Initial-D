# Landing Page de Initial D

Este proyecto es una landing page de una serie animada llamada Initial D. La landing page está diseñada como parte de un trabajo práctico de Interacción con Dispositivos Móviles, realizado en la Facultad Da Vinci, bajo la supervisión del profesor Alejandro D'ADDEZIO.

## Serie Animada: Initial D

Initial D es una serie de anime y manga que se centra en las carreras callejeras de coches en las montañas de Japón. La trama sigue a Takumi Fujiwara, un repartidor de tofu con habilidades de conducción impresionantes, mientras se enfrenta a desafíos en el mundo de las carreras ilegales.
